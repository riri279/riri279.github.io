    EngHacks 2017

    Objective/Problem:

    At this hackathon, we created a device to help people who are visually impaired navigate the obstacles around them.

    Our Solution:

    To improve the user experience we identified user needs and the best way to deliver information to visually impaired users. Then we used open CV, numpy, and python 2.7 to recognize obstacles and deliver audio output.

    Takeaways: Difficulties and learnings

    I came into this hackathon expecting to develop a software application, but it gave me the opportunity to apply my UX knowledge to a hardware project which is pretty exciting! Since I had no programming background previous to this hackathon, I didn't have much knowledge of any of those programs so when I managed to use python 2.7 to the create audio messages that users would hear, I was pretty happy!
